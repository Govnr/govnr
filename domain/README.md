# Domain

Pure Govnr civic domain.

This layer owns entities, value objects, policies, domain events, and invariants for groups, membership, petitions, motions, voting, decisions, and civic records.

Domain documentation belongs here, alongside the model it describes. Use this area for Govnr civic concepts, terminology, lifecycle rules, modelling decisions, and ubiquitous language.

Start with [DOMAIN_SPECIFICATION.md](DOMAIN_SPECIFICATION.md).

Rules:

- No HTTP framework imports.
- No database imports.
- No queue, AI, email, or object-storage SDK imports.
- Keep civic rules explicit and testable.
